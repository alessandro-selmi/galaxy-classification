# Galaxy Morphology Classification: From Hand-Crafted Features to Deep CNNs

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/alessandro-selmi/galaxy-classification/blob/main/Galaxies.ipynb)

Comparative study of classical machine learning and deep learning models for galaxy morphology classification on a dataset of 11k RGB images.
Implemented feature extraction pipelines and trained logistic regression, MLP and CNN models in PyTorch (ResNet18 transfer learning and data augmentation) and additional classical models using scikit-learn, achieving 98.9% test accuracy.

The project analyses a dataset of 11k 224x224px RGB astronomical images (galaxies) initially provided as part of the coursework for Machine Learning, University of Pavia. The initial version was developed within that course and then expanded into the current version, autonomously developed for the Artificial Intelligence for experimental and Applied Physics course, University of Pavia.
