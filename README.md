# Galaxy Morphology Classification: From Hand-Crafted Features to Deep CNNs

Comparative study of classical machine learning and deep learning models for galaxy morphology classification on a dataset of 11k RGB images.
Implemented feature extraction pipelines and trained logistic regression, MLP and CNN models in PyTorch (ResNet18 transfer learning and data
augmentation) and additional classical models using scikit-learn, achieving 98.9% test accuracy.
